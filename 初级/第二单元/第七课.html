语法

宾格助词を

接续：宾语 (体言) + を + 动词 (一般接他动词)

他动词：动词需要一个宾语才能完整地表现主语的动作或作用的词。相当于英语的及物动词。

自动词：动词本身能完整地表示主语的某种动作的词。前面一般是不加「を」的。相当于英语的不及物动词。

日语中句子的构成：主语 + 宾语 + 谓语

  • 手を洗う (あらう)
  • 物を買う
  • 新聞を読む
  • テレビを見る
  • 私は毎日６時に起きる。
  • 私は毎晩テレビを見ます。
  • 王さんは毎朝新聞を読みます。
  • テニスをする テニスする -> X
  • 勉強をする = 勉強する

格助词で

用法1：

接续：交通工具 + で，含义：利用 ... 交通工具

  • 自転車で学校へ行きます。

用法2： 接续：场所名词 + で，含义：动作发生的场所

  • 李さんはいつも図書館で本を読みます。
  • 学生は教室で勉強します。
  • 学生は教室で勉強をします。
  • 王さんは居間でテレビを見ます。

に：用于存在句中

で：动作发生的场所

并列助词か

か (终助词)

接续：放在句末，含义：表示疑问

  • あなたは学生ですか。

か (并列助词)

接续：体言/活用词终止形 + か，含义：表示选择

活用词：可以变化的词，终止形：结句的形式

  • 自転車で行くか、バスで行くか、早 (はや) く決 (き) めてください。
  • 李さんは朝パンかまんじゅうを食べます。
  • コーヒーかお茶をください。
  • コーヒーとお茶をください。
  • 図書館と映画館へ行きます。
  • 図書館か映画館へ行きます。

基本课文

B

甲：今朝何を食べましたか。

何：在で，の前面一般念成「なん」，其他助词前一般念成「なに」

乙：何も食べませんでした。

特殊疑问词 + も + 肯定，表示全盘肯定

特殊疑问词 + も + 否定，表示全盘否定

特殊疑问词 + でも + 肯定，表示全盘肯定

  • 何でも分かります。

C

甲：吉田さん、日曜日何をしますか。

乙：テニスかジョギングをします。

在体育方面，「を」是不可以省略的。

  • 勉強をする
  • 勉強する
