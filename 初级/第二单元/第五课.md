## 语法
### 动词的种类及变形
#### 动词：词干 + 词尾 (う段)

書く、賑やかだ、忙しい、勉強する、来る、起きる、寝る、立つ、洗う、これ、呼ぶ、読む、話す

其中不是动词的有：賑やかだ、忙しい、これ

#### 动词的分类
##### 五段动词
1. 不是以「る」结尾的：書く
2. 以「る」结尾的，「る」前面的一个假名不是「い」段或「え」段的：終わる
3. 特殊的五段动词：走 (はし) る、帰 (かえ) る、入 (はい) る等

##### 一段动词
以「る」结尾的，「る」前面的一个假名是「い」段或「え」段的。

上一段动词：「る」前面的一个假名是「い」段的，如：起きる

下一段动词：「る」前面的一个假名是「え」段的，如：寝る

##### 不规则动词
###### サ变动词「する」
1. 「する」作为实义动词
	* テニスをする
	* 勉強をする
2. 动作性名词 + する，使名词变成动词
	* 勉強する：サ变动词
	* 勉強をする：「勉強」是名词
	* 散歩 (さんぽ) する
	* 散歩をする

###### ヵ变动词「来る」
#### 动词的连用形I (ます形)
五段：う -> い + ます

* 書く -> かき + ます
* 買う -> 買い + ます

一段：去る + ます

* 食べる -> 食べ + ます

不规则：

* サ变する -> し + ます
	- 勉強する -> 勉強し + ます
* ヵ变：来 (く) る -> 来 (き) + ます

### 助动词ます
接续：动词的ます形 + ます

* 简体：勉強する
* 敬体：勉強します

时态：

1. 表示将来，意志
	* 私は日本へ行きます。(行く)
2. 客观，恒常
	* 春 (はる) になると、花が咲 (さ) きます。(咲く)
3. 经常，反复的动作
	* 毎日、六時に起きます。(起きる)

否定

* ます -> ません
	- 王さんは日本へ行きません。
	- 私は帰りません。
* ました -> ませんでした
	- 去年王さんは日本へ行きました。
	- 去年王さんは日本へ行きませんでした。
	
### 格助词に
接续：具体时间 + に

「明日」、「先週」、「去年」等笼统的时间是不能加「に」的。

* 私は夜１１時３０分に寝ます。
* 授業は午前８時に始まります。
* 毎朝６時に起きます。

### ~ から ~ まで
接续：体言 + から + 体言 + まで

から：时间，空间的起点

まで：时间，空间的终点

* 家から学校までバスで１時間ぐらいかかります。
* 毎晩６時から８時までテレビを見ます。
* 私は中国から来ました。
* 授業は夜９時までです。
