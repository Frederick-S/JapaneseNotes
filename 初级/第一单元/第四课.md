## 语法
### ~ は ~ にあります (ありません)
表示人或动物以外的无生命体的存在。

あります：原形「ある」，否定为「ありません」。

接续：体言 + は + 场所名词 + にあります (ありません)

に：事物存在的场所。

例子：

* 王さんの家は一階にあります (ありません)。
* バナナはテーブルの上にありません (あります)。
* ペンは机の上にあります。

### ~ は ~ にいます (いません)
表示人或动物的存在。

います：原形「いる」，否定为「いません」。

接续：体言 + は + 场所名词 + にいます (いません)

に：事物存在的场所。

例子：

* 学生は教室にいます (いません)。
* 猫は机の下にいます (いません)。
* かばんは机の下にいません。(错误)
* かばんは机の下にありません。(正确)

总结：

あります：表示人或动物以外的无生命体的存在，包括植物。

います：表示人或动物的存在。

### ~ に ~ が (も) あります (ありません)
接续：场所名词 + に + 体言 + が (も) あります (ありません)

に：事物存在的场所。

表示人或动物以外的无生命体的存在。

例子：

* 机の上にすいかがあります。
* 机の上にりんごもあります。
* 部屋に机があります。
* 部屋にテーブルもあります。
* 引き出し (ひきだし) の中にかばんがありますか。

### ~ に ~ が (も) います (いません)
接续：场所名词 + に + 体言 + が (も) います (いません)

に：事物存在的场所。

表示人或动物，有生命体的存在。

例子：

* 教室に学生がいます (いません)。
* 机の下に猫もいます (いません)。
* 机の下に本がいます。(错误)
* 机の下に本があります。(正确)

### ~ には ~ はありません/いません
### ~ には ~ があります/います

接续：场所名词 + には + 体言 + はありません/いません

场所名词 + には + 体言 + があります/います

「~ には」的「は」：强调，对比

「~ はありません/いません」的「は」：引出否定，表强调

例子：

* 王さんの家にはバナナ**は**ありません。
* 王さんの家にはバナナ**が**ありません。
* 王さんの家**に**バナナはありません。
* 部屋**には**エアコンがあります。
* 部屋**には**猫がいます。
* 部屋**に**猫がいます。

### 并列助词「と」
体言 + と + 体言

表示完全列举

* 私と李さんは友達 (ともだち) です。
* それは李さんと王さんの本です。
* 机の上に本とノートがあります。
* 王さんと張さんは教室にいます。

### 助词「も」
(1) 接续：体言 + も，表示兼提。

王さんは学生です。私も学生です。

(2) 接续：特殊疑问词 + も，表示全盘肯定或全盘否定。

* 誰もいません。
* 何 (なに) もありません。
