## 语法
### ~ は ~ です
接续：场所代词 + は + 体言 + です。

场所代词：ここ/そこ/あそこ、こちら/そちら/あちら

体言 + は + 场所代词 + です。

* ここは銀行です。
* 銀行はここです。
* そこは病院です。
* 病院はそこです。
* あそこはホテルです。
* ホテルはあそこです。

### ~ は ~ ですか
接续：场所代词 + は + 体言 + ですか。

体言 + は + 场所代词 + ですか。

* ここは銀行ですか。
* 病院はそこですか。
* あそこはホテルですか。

总结：

* これ提问 -> それ回答
* それ提问 -> これ回答
* あれ提问 -> あれ回答

注意：この/その/あの + 体言

* ここ提问 -> ここ回答
* そこ提问 -> そこ回答
* あそこ提问 -> あそこ回答

例子：

* ここは教室 (きょうしつ) ですか。はい、ここは教室です。/ はい、そうです。
* 浴室 (よくしつ) はそこですか。いいえ、浴室はそこではありません。/いいえ、違います。/いいえ、そうではありません。

### ~ は + 特殊疑问词 + ですか
特殊疑问词：何、どなた、どれ、だれ等。

接续：体言 + は + 特殊疑问词 + ですか。

* ホテルはどこですか。
* 王さんは誰ですか。

接续：特殊疑问词 + は + 体言 + ですか。(**错误**)

特殊疑问词 + **が** + 体言 + ですか。(回答用 が)

* どこがホテルですか。ここがホテルです。

### 提示助词「も」
接续：体言 + も，相当于汉语的 "也"。

* 王さんは学生です。私も学生です。
* それは私の本です。これも私の本です。
* ここはホテルです。そこもホテルです。

### ~ は ~ ですか ~ ですか
接续：体言 + は + 体言 + ですか、体言 + ですか，表示答案有多种可能，而询问其中的一种。

选择疑问句不能用はい或いいえ回答。

* ここは学校ですか、病院ですか。ここは学校です。
* あの人は王さんですか、李さんですか。あの人は王さんです。
* これは王さんの服ですか、張さんの服ですか。それは王さんの服です。
