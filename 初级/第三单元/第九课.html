语法

形容词做谓语 (形容词放在句子的最后)

形容词特点：

  1. 形容词是说明事物性质状态的词
  2. 形容词构成：词干 + い
  3. 形容词是活用词，词尾「い」有时会发生变化

形容词做谓语，形容词放在句子的最后，形成 "描写句"。

句型：... は ... です。

描写句子里的「です」没有实质意思，可省去。加上「です」表达的感觉更加客气。

(1) 现在时形容词做谓语的肯定形式

体言 + は + 形容词 (原形) + です (「です」可省去)

  • この花は赤いです。
  • この花は赤い。
  • その建物は高いです。
  • その建物は高い。

(2) 现在时形容词做谓语的否定形式

变形：

形容词词尾「い」变成「く」后面加「ない (です)」(「ない」为形容词)，或形容词词尾「い」变成「く」后面加「ありません」。

「いい」的变形要用「よい」来变形。

  • よくない。
  • よくないです。
  • よくありません。
  • これは高いです。
  • これは高くないです。
  • これは高くありません。
  • 近くありません。
  • 近くないです。
  • 天気はいい/よいです。
  • 天気はよくないです。
  • 天気はよくありません。

(3) 过去时形容词做谓语的肯定形式

形容词的过去时表示：

  1. 过去的情况
  2. 感慨，感叹

变形：形容词词尾「い」变成「かった」加上「です」(简体形式可省去「です」)

  • よい -> よかった (です)
  • 悪い -> 悪かった (です)
  • 昨日の天気は悪かったです。

(4) 形容词做谓语时的过去否定形式

变形：

形容词词尾「い」变成「く」加上「なかった (です)」，或者形容词词尾「い」变成「く」加上「ありませんでした」。

  • 昨日は暑くなかった。
  • 昨日は暑くなかったです。
  • 昨日は暑くありませんでした。
  • 昨日は寒くありませんでした。

形容词做定语 -- 形容词修饰体言

体言修饰体言：体言 + の + 体言

形容词修饰体言：形容词原形 (形容词い形) + 体言

  • 赤い花
  • 高い建物
  • 古い友達
  • いい/よい天気

あまり + 否定形式，表示 "不太，不怎样"

(注：口语中[あまり]常会音变成「あんまり」)

  • あまり行きません。
  • 私はあまりテレビを見ません。
  • あまり高くありません。
  • あまり高くないです。
  • この西瓜 (すいか) はあまり甘くありません。

「を」->「は」/「も」

「は」做 "提示助词"，可以 "提示主题"，强调对比，引出否定

「も」做 "提示助词"，相当于中文的 "也"

「に」、「へ」、「で」可以和「は」/「も」重叠使用

「を」不能和「は」/「も」重叠使用，一般用「は」/「も」来替换「を」

  • バナナを食べます。みかんを食べません。
  • バナナは食べます。みかんは食べません。
  • りんごを食べます。
  • りんごも食べます。

注意：

「を」/「が」碰到「は」/「も」时，一般用「は」/「も」来取代「を」/「が」，相应的句意也会发生变化。
