## 语法
### 动词可能态
可能态变形：

#### 五段动词
古代变法：五段动词词尾落在「ウ」段上的假名 -> 该假名所在行的「あ」段 +「れる」

现代变法：五段动词词尾落在「ウ」段上的假名 -> 该假名所在行的「え」段 +「る」

* 書く (かく)，古代：かかれる，現代：かける
* 買う (かう)，古代：かわれる，現代：かえる

#### 一段动词
去「る」+「られる」

食べる -> 食べられる

#### サ变动词
する -> できる 

勉強する 

#### カ变动词
来る -> 来られる

注意：可能动词都是一段动词

意义：可能动词表示有能力做 ...，有条件做 ...，"能 ~"，"会 ~"

* 書く -> 書ける (能写，会写)
* 買う -> 買える (能买)
* 食べる -> 食べられる (能够吃)
* 勉強する -> 勉強できる (能够学习)
* 来る -> 来られる (能够来)

句型：

1. ... は ... が ... れる/られる，可能动词前面用「が」，强调对象语
2. ... は ... を ... れる/られる，用「を」强调动词 (此句型很少见)

* 李さんは日本語を話します。
* 李さんは日本語が話せます。(强调对象语)
* もう3年間日本語を勉強したのに、言いたいことをなかなか話せなくて、悔しいと思います。(侧重动词，这种情况较少用)
* 寒い冬の朝はなかなか起きられません。
* 明日用事があるので、来られません。
* 11時になっても、寝られません。
* 11時になっても、眠れません。

「... ことができる」与动词可能态的区别

1. ... ことができる：多指由于外部原因，客观原因造成的能与不能。
2. 动词的可能态：多指由内部、个人的原因，或主观原因造成的能与不能。

* 芝生の中に入ることができません。(客观原因)
* 彼は体が大きいですから、その穴に入れません。(个人原因)

注意惯用句型：

* 手に負えない。(处理不了)
* 手に負うことができない。(错误)

「見える」「見られる」的区别：

1. 見える：多用于由于眼力、空间距离等客观因素的 "可以看到"。
	* 黒板の字は小さいですから、見えない。
	* 雨が止んで、光が雲の隙間 (すきま) から見えた。
2. 見られる：侧重在某些特定条件下 "可以看到"。
	* 自動販売機は日本の至る所で見られる。(在其他国家可能看不到)

 注意点：「聞ける」「聞こえる」之间的区别和「見える」「見られる」之间的区别基本相同。

### ... ように
接续：动词的基本形 (无意志动词) + ように，... ない ＋ ように

意义：为了实现前项而努力做后项 (注：后项无法控制前项) "为了 ..."

* 授業に遅れないように、早く家を出ました。
* 魚が買えるように、列に並んだ。
* みんなが見えるように、字を大きく書きました。
* 風を吹き込 (こ) まないように、窓を閉めなさい。

ように和ために的区别：

1. 接续
	* ように：动词基本形 + ように (无意志动词)，... ない ＋ ように
	* ために：动词基本形 + ために (有意志动词)，名词 + の ＋ ために
2. 意义
	* ように：后项无法控制前项
	* ために：后项可以控制前项

例子：

* 魚を買うために、列に並んだ。
* 魚が買えるように、列に並んだ。
* 彼は日本へ留学に行くために、一生懸命勉強します。(有意志动词，后项可以控制前项)
* 試験が合格できるように、一生懸命勉強します。(无意志动词，后项无法控制前项)

### ... ようにする
接续：动词基本形 + ようにする，~ ない + ようにする

意义："尽量做 ..."，"努力做 ..."，经常与「できるだけ」、「なるべく」一起用

* これから、授業に遅れないようにしています。
* 用事がなければ、外出しないようにしています。
* 毎晩10時ごろ寝るようにしています。

### ... ようになる
复习：「なる」表示自然的变化　

接续：

1. 形容词把「い」变「く」+「なる」
	* 北海道は10月になると、寒くなります。
2. 形容动词词干 +「に」+「なる」
	* 放課後、学校は静かになります。
3. 名词 +「に」+「なる」    
	* 春になると、花が咲きます。

新语法：「... ようになる」

接续：动词基本形 + ように + なる 

意义：表示动作状态的自然变化，养成一种新的习惯

* 弟は以前あんまり勉強しなかったんですが、大学に入ってから、よく勉強するようになりました。
* 兄は最近お酒を飲むようになりました。
* あの人は最近早く起きるようになりました。

注：动词的否定 +「ようになる」为：动词的未然形I形 +「ない」+「ようになる」

妹は最近あまり遊ばないようになりました。
